# 自定义图标

本地图标:

在.vuepress目录下新建icons目录用于存储 Svg 图标,图标前缀为icon-

推荐使用阿里巴巴失量图标库 (opens new window),支持Unicode、Font class和Symbol。请参考帮助文档 (opens new window)

# 使用图标

可直接在 vue 、 markdown 中使用图标

<vp-cion name="home"/>

# 选项

# name

  • 类型： string
  • 必须: true 图标名称

# size

  • 类型： string
  • 必须: false 图标大小

# color

  • 类型： string
  • 必须: false 图标颜色
<vp-cion name="home" size="1rem" color="#fff"/>
